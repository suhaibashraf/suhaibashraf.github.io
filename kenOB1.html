RoboGuard KenOB1

Home KenOB1

RoboGuard KenOB1 is an autonomus security human size robot developed by OneBerry Technologies, Singapore.

I have upgraded this system from 2d Sick Lidar to 3d velodyn Lidar and depth camera. I have used ROS(C/Python), Roboteq, Depth Camera (Realsense D435i, Zed mini), Velodyne Puck(VLP-16), Rtab, Hdl graph slam and Teb local path planner.

The main goal of this project was to improve the performace of KenOB robots and extend its limitations.
